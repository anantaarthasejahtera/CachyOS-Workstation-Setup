# 🌟 Introduction

Welcome to the official documentation for the **CachyOS Workstation Setup**.

This project provides a **modular installer** that transforms a fresh CachyOS installation into a fully configured, aesthetically stunning developer workstation. It auto-detects your hardware (GPU, RAM, CPU cores, Secure Boot) and adapts accordingly.

### 🛡️ Safety First

This installer is **safe on existing systems**. All configurations are automatically backed up to `~/.config-backup/` before any changes are made, allowing you to easily roll back anything using our built-in Time Machine workflow.
